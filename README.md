
# Pillars

This library is a basis for backend applications written in Scala 3 using the TypeLevel stack.


![Logo](docs/images/logo.png)


## Features

- Admin server
- Configuration
- Database access
- Feature flags
- Logging
- OpenTelemetry-based observability
